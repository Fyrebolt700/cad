# Assembly Procedure

## Project
Robotic Actuator Validation Fixture

## Purpose
This procedure describes how to assemble the actuator validation fixture from the CAD-designed components. The fixture is intended to support future actuator endurance testing with adjustable loading, sensor mounting, and safety guarding.

## Components

- Base plate
- Actuator mount
- Load arm
- Adjustable weight block
- Sensor bracket
- Safety guard
- Rotary actuator or servo motor, purchased component, not modeled in this CAD revision
- Position sensor, purchased component, not modeled in this CAD revision
- M3/M4 fasteners

## Procedure

### 1. Prepare the base plate
Place the base plate on a flat work surface. Confirm that all mounting holes and slots are clear.

### 2. Install the actuator
Mount the rotary actuator or servo below the base plate. Align the actuator shaft with the center clearance hole in the base plate.

### 3. Attach the load arm
Attach the load arm to the actuator shaft above the base plate. Confirm that the arm can rotate freely without contacting the base plate.

### 4. Install the adjustable weight block
Place the weight block on one of the load arm holes. Use the hole position required for the test load condition. Secure it with a fastener.

### 5. Install the sensor bracket
Mount the sensor bracket to the adjustable slots on the base plate. Position the bracket close to the load arm path for future position or limit sensing.

### 6. Install the sensor
Attach the selected position sensor, limit switch, or encoder placeholder to the vertical slots on the sensor bracket. Adjust sensor height and position as needed.

### 7. Install the safety guard
Place the safety guard over the moving load arm area. Confirm that it covers the main sweep region of the arm without interfering with motion.

### 8. Final inspection
Check that:
- The load arm rotates freely
- The weight block is secured
- The sensor bracket is adjustable
- The safety guard does not interfere with motion
- All fasteners are tightened
- Wiring paths are clear of moving parts

## Notes
This is a CAD design project for a future prototype. The actuator, position sensor, and DAQ hardware are treated as purchased components and are not modeled in this CAD revision.
